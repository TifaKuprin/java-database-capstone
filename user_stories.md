# Smart Clinic Management System - User Stories

## Doctor

- As a doctor, I want to view my daily appointments so that I can manage my schedule.
- As a doctor, I want to see patient details so that I can provide proper treatment.
- As a doctor, I want to view my available time slots so that patients can book appointments.

## Patient

- As a patient, I want to search doctors by name or speciality so that I can find the right doctor.
- As a patient, I want to book an appointment with a doctor so that I can receive medical consultation.
- As a patient, I want to view my upcoming appointments so that I don't miss them.

## Admin

- As an admin, I want to add doctors to the system so that patients can book appointments.
- As an admin, I want to manage users so that system access is controlled.
- As an admin, I want to monitor appointments so that clinic operations run smoothly.
